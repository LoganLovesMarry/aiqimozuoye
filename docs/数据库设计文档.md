# 数据库设计文档

## 数据库名：employment_db

## 核心表结构
### student表
- id, name, email, password, major, resume_url, status, create_time

### company表
- id, name, email, password, license, status, create_time

### job表
- id, title, description, salary, location, company_id, status, create_time

### application表
- id, student_id, job_id, status, apply_time

### admin表
- id, username, password, role
