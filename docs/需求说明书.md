# 毕业生就业信息管理系统 - 需求说明书

## 1. 项目背景
高校毕业生就业管理信息化程度低，需构建统一平台，实现学生求职、企业招聘、学校管理三位一体。

## 2. 用户角色
- 学生
- 企业HR
- 高校就业管理员

## 3. 功能需求
### 学生模块
- 注册/登录
- 简历上传与管理
- 职位搜索与投递
- 就业状态跟踪

### 企业模块
- 注册/认证
- 职位发布
- 简历筛选
- 面试通知与反馈

### 管理员模块
- 用户审核
- 就业数据统计
- 公告发布
- 权限管理

## 4. 非功能需求
- 性能：支持500+并发
- 安全：密码加密、权限控制
- 可用性：7×24小时
- 可扩展：支持AI推荐、移动端
