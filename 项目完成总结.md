## 如何运行项目

### 快速启动三步骤

1. **初始化数据库**
   ```bash
   mysql -u root -p < sql/employment_db.sql
   ```

2. **配置数据库连接**
   修改 `backend/src/main/resources/application.yml` 中的数据库配置

3. **启动项目**
   ```bash
   cd backend
   mvn spring-boot:run
   ```
   或者在IDE中运行 `EmploymentApplication.java`

### 访问地址
- 后端API: http://localhost:8080/api

## 注意事项

### IDE警告说明
你可能会看到IDE中显示包名错误的警告：
```
The declared package "com.employment" does not match the expected package "main.java.com.employment"
```

**这是正常现象！**原因：
- Maven项目的源代码根目录是 `src/main/java`
- IDE在Maven项目加载前，会临时显示此警告
- 解决方法：在IDE中刷新Maven项目即可

在IntelliJ IDEA中：
1. 右键项目 → Maven → Reload Project
2. 或者点击Maven工具窗口的刷新按钮

这些警告不影响项目编译和运行。
