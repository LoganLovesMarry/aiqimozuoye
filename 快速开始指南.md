# 快速开始指南

## 第一步：安装必要软件
1. 安装 **JDK 8** 或更高版本
2. 安装 **Maven 3.6+**
3. 安装 **MySQL 8.0**
4. 安装 **IntelliJ IDEA** 或 **Eclipse**（推荐IDEA）

## 第二步：初始化数据库
1. 启动MySQL服务
2. 使用MySQL客户端或Navicat连接到MySQL
3. 执行SQL脚本：
   ```sql
   source E:/idea-workspace/aiqimozuoye-1/sql/employment_db.sql
   ```
   或者直接在MySQL客户端中打开并运行该SQL文件

## 第三步：配置数据库连接
修改配置文件：`backend/src/main/resources/application.yml`

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/employment_db?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai&useSSL=false
    username: root      # 改为你的MySQL用户名
    password: root      # 改为你的MySQL密码
```

## 第四步：导入项目到IDE

### 使用IntelliJ IDEA（推荐）
1. 打开IDEA
2. 选择 **File** -> **Open**
3. 选择 `E:\idea-workspace\aiqimozuoye-1\backend` 目录
4. 点击 **Open as Project**
5. 等待Maven自动下载依赖（首次可能需要较长时间）
6. 依赖下载完成后，IDE会自动识别项目结构

### 使用Eclipse
1. 打开Eclipse
2. 选择 **File** -> **Import** -> **Maven** -> **Existing Maven Projects**
3. 选择 `E:\idea-workspace\aiqimozuoye-1\backend` 目录
4. 点击 **Finish**
5. 等待Maven自动下载依赖

## 第五步：运行项目

### 方法一：通过IDE运行
1. 找到 `com.employment.EmploymentApplication` 类
2. 右键点击该文件
3. 选择 **Run 'EmploymentApplication'**
4. 等待项目启动，看到如下日志表示启动成功：
   ```
   Started EmploymentApplication in X seconds
   ```

### 方法二：通过Maven命令行运行
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

## 常见问题

### 1. 端口被占用
如果8080端口已被占用，修改 `application.yml` 中的端口：
```yaml
server:
  port: 8081  # 改为其他端口
```

### 2. 数据库连接失败
- 检查MySQL服务是否启动
- 检查用户名密码是否正确
- 检查数据库是否已创建
- 确认MySQL端口为3306

### 3. Maven依赖下载失败
- 检查网络连接
- 配置Maven镜像（阿里云镜像）
- 清除本地Maven缓存：`mvn clean`

### 4. IDE中包名显示为红色
这是正常现象，Maven项目的包结构是 `com.employment`，不包含 `main.java`。
只要能正常运行即可，或者：
1. 在IDEA中：右键项目 -> **Maven** -> **Reload Project**
2. 在Eclipse中：右键项目 -> **Maven** -> **Update Project**

### 5. 文件上传失败
确保项目有创建 `uploads/resumes/` 目录的权限。
